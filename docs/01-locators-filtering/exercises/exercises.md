# Вправи — Модуль 01: Locators та фільтрація

## Вправа 1: Перепиши локатори на `getByRole`

**Завдання:** візьми будь-які 3 CSS/XPath локатори з тесту та перепиши їх на `getByRole()` (або інші recommended локатори).

**Критерій:** локатор має бути читабельним і стійким до змін верстки.
